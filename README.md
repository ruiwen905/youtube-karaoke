###Getting Started :bulb:

Install in your localhost do the following :octocat:
```
> git clone https://github.com/ruiwen905/youtube-karaoke.git youtube-karaoke
> cd youtube-karaoke
> npm install
> npm start
> Go to localhost:3000
```
